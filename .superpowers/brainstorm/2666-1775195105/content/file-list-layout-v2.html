设计要点

  • 密级仅保留"公开"：所有文件卡片中密级标签统一显示为绿色的"公开"，后续如需扩展可通过配置添加
  • 左侧标签树：按文档类型大类分组，每类下展开对应的标签字段及可选值，点击筛选右侧文件
  • 右侧卡片列表：每个文件一张卡片，显示文档类型标签、密级、标题、作者、摘要、标签值、文件大小
  • 标签药丸样式：用圆角小标签显示文件关联的标签值，不同文档类型用不同颜色
  • 顶部搜索栏：支持按名称/作者/摘要搜索，按文档类型筛选
  • 操作按钮：每张卡片右下角有下载、编辑、删除操作
