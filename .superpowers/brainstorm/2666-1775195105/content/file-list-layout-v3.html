涡扇发动机高温合金叶片疲劳寿命评估方法研究
作者张三, 李四 技术体系航空发动机 来源中国航发研究院 上传2024-03-15
本报告针对涡扇发动机高压涡轮叶片在复杂热力载荷下的疲劳寿命问题，提出了一种基于损伤力学和有限元仿真相结合的评估方法。通过建立叶片三维有限元模型，考虑温度梯度、离心力和气动载荷的耦合作用，对叶片的蠕变-疲劳交互作用进行了系统的参数化研究。

一种航空发动机叶片冷却结构及冷却方法
作者赵六, 钱七 技术体系航空发动机 来源本单位 上传2024-01-10
本发明涉及航空发动机技术领域，公开了一种涡轮叶片复合冷却结构，包括气膜冷却孔、冲击冷却通道和尾缘劈缝。通过优化冷却通道的布置方式和冷却气流的分配比例，显著提高了叶片的冷却效率和服役寿命。
